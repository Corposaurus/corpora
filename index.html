Welcome to our directory of biomedical corpus.

We compiled the BibTex citations for all corpora.

We include readers for all corpora and writers for some popular formats, such as BioC, PubAnnotation, as well as the plain text WBI format.

The table below lists all corpora compiled in our directory and for which format we provide support.
